I have started learning web development. This is my javascript based mini-project. 
I have implemented a web based rock paper scissors game using html, css, javascript. 
I have used concepts like DOM manipulation, event handling, random numbers, conditionals and much more.
In this project, you can play the game with the computer infinitely. The game checks for a winner after every move and also detects if a draw takes place. It is a complete mini-project, that handles inputs and shows proper outputs.
